# テンプレート

> これらのテンプレートを使用して、Braze Docs で[pages]({{site.baseurl}}/contributing/content_management/pages/) または[sections]({{site.baseurl}}/contributing/content_management/sections/) を作成できます。

テンプレートの各セクションの詳細については、以下のようなHTML コメントを参照してください。

\`\`\`マークダウン
<!-- Here's an HTML comment! -->
```

{% alert important %}
これらのコメントは、書き込み中にファイルに保持できますが、公開する前に削除する必要があります。
{% endalert %}
